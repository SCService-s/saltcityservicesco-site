# Salt City Services Co — Website

One-page consulting site for saltcityservicesco.com. Built with plain HTML + CSS, no build step required.

## How to edit

Open `index.html` in any text editor (or directly in GitHub's web editor). All content and styles are in this one file.

To find a section quickly, search for:
- `<!-- HERO -->` — main headline and subheadline
- `<!-- PROBLEM -->` — the 3 pain point cards
- `<!-- SERVICES -->` — the 3 service cards
- `<!-- PROOF -->` — the 4 outcome bullets
- `<!-- ABOUT -->` — bio and certifications
- `<!-- CONTACT -->` — contact buttons and email address

## Updating the Credly cert links

In the About section, replace the `href="https://www.credly.com"` placeholders with your actual Credly badge URLs. Find them at credly.com → your profile → each badge → Share.

## Deployment

This site is hosted on GitHub Pages. Push any change to the `main` branch and it goes live within ~60 seconds.

## Domain

`saltcityservicesco.com` is connected via the `CNAME` file and DNS A records at the registrar.
